# AI Dev Showcase

> **「課題をリプライするだけで、AIが動くシステムを作ります」**

Xで開発課題を募集し、Claude Code（AIエージェント）がリアルタイムで実装・デプロイするプロジェクトのショーケースです。

---

## 仕組み

```
[X] 毎日、開発課題を募集するツイートを投稿
    ↓
[あなた] 「こんなシステムが欲しい」をリプライ
    ↓
[Claude Code] 課題を分析して実装
    ↓
[Vercel] デプロイ → 7日間限定で公開
    ↓
[GitHub] コードを永続公開（このリポジトリ）
```
